# 03 — Frontend (Next.js — маркетинг и CRM UI)

## Структура

- `apps/web` — публичный маркетинговый сайт (Next.js App Router + SSG/ISR/CSR для SEO и интерактивных страниц)  
- `apps/crm` — SPA/SSR интерфейс для порталов (Next.js, React Server Components для Dashboard)  
- Shared UI: `packages/ui` (shadcn/ui + Tailwind + компоненты под CRM и маркетинг)  
- Shared utils: `packages/hooks`, `packages/types`, `packages/api-client`  

---

## Главные страницы маркетингового сайта

### 1. Главная страница
- Hero блок с видео/изображением  
- Call-to-action кнопки («Записаться», «Посмотреть каталог мастеров»)  
- Интеграция с AI: динамически рекомендованные статьи/услуги  
- SEO‑метаданные, OG/Twitter карточки, schema.org  

### 2. Каталог мастеров / поиск по городу
- Фильтры: город, стиль тату, рейтинг, цена  
- Сортировка: по рейтингу, цене, популярности  
- Профили мастеров: портфолио, отзывы, доступные слоты  
- Lazy load / infinite scroll  

### 3. Новости / блог
- SEO оптимизация: headings, metadata, internal linking  
- Автогенерация AI-статей и FAQ  
- Компоненты: статьи, карточки постов, категория и теги  
- Комментарии (опционально)  

### 4. Страницы услуг (landing)
- Детали услуги, стоимость, описание, портфолио  
- Call-to-action: запись на услугу  
- Schema.org для услуг и цен  

### 5. Контакты / форма записи (embed form)
- Интерактивная форма для лидов → отправка в CRM через API  
- Валидация форм и защита от спама (reCAPTCHA)  
- Подтверждение записи email/SMS  

### 6. Личный кабинет клиента
- Просмотр записей, историю взаимодействий  
- Возможность загружать референсы (фото)  
- Настройки профиля и уведомлений  

---

## CRM UI (apps/crm)

### 1. Dashboard
- Основные KPI: новые лиды, записи, активные сделки, revenue  
- Графики и диаграммы (chart.js / recharts)  
- Быстрые действия: «Добавить клиента», «Создать сделку»  

### 2. Сущности CRM
- **Клиенты (Client)**: карточка клиента, заметки, файлы, история взаимодействий  
- **Лиды (Lead)**: источник, статус, ответственный менеджер, напоминания  
- **Сделки (Deal)**: этапы воронки, прогнозируемый доход, связанные встречи  
- **Календарь**: интеграция с Google Calendar, drag&drop, запись на слот
