# 07 — User Account / Portal onboarding

## Функции регистрации
- Регистрация портала (owner) — email + подтверждение
- Приглашение сотрудников по email
- Social login (Google) для упрощённой регистрации
- Подтверждение номера телефона (SMS) — опционально

## Безопасность
- Email verification
- 2FA (TOTP) — опционально для owner
- Refresh tokens + device management
- Session management и возможность logout всех сессий

## Клиентская часть (клиенты салона)
- Личный кабинет: записи, обратная связь, отзывы
- Быстрая запись на сессию
- Хранение референсов (фото)
- GDPR: право на удаление данных
