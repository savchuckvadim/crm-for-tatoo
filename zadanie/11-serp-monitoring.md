# 11 — SERP Monitoring

## Описание
- Daily cron → SerpAPI → сохраняет топ N результатов по ключам, на которые таргетится проект
- Для каждого портала/маркетингового сайта отслеживаются позиции

## Бизнес-логика
- Если падение > threshold (например 3 позиции) → создаётся задача на анализ
- Анализ делает AI (см. SeoModule) и публикует рекомендации в админку + уведомляет в Telegram/Email

## Хранение данных
- serp_positions (keyword_id, date, position, snippet)
- history retention — 1 год (архивировать дальше)
