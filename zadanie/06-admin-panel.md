# 06 — Admin & CRM Panel

## Роли и права
- Owner: полный доступ к порталу и оплатам
- Manager: управление пользователями, сделками
- Artist: профиль мастера, запись клиентов
- Accountant: доступ к заказам/финансам

## CRUD для сущностей
- Клиенты: карточка клиента, история, заметки, файлы (тату‑референсы)
- Лиды: источник, статус, назначенные менеджеры
- Сделки: этапы воронки, прогноз, связанные appointment
- Кастомные поля: UI для создания и редактирования полей
- Импорт/экспорт CSV

## Удобства
- Быстрая запись из формы на сайте → лид в CRM
- Drag&drop портфолио
- Быстрые шаблоны писем/SMS
- Массовые операции (merge clients, bulk change status)
