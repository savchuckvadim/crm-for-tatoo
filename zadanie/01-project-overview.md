# 01 — Project Overview

**Название:** CRM для тату‑салонов (monorepo: CRM + маркетинговый сайт)

**Цель проекта:**  
Создать масштабируемую, безопасную и SEO‑оптимизированную платформу, которая объединяет:  
- полнофункциональную CRM систему для тату‑салонов (порталы/салоны/сотрудники),  
- маркетинговый сайт (Next.js) для продвижения услуг и привлечения клиентов,  
- автоматизированную SEO/маркетинг‑машину на базе AI, которая помогает продвигать сайт без постоянного ручного участия SEO‑специалистов.  

---

## Основные сценарии использования

1. **Владелец салона / администратор портала**  
   - регистрирует портал (subdomain или в рамках домена), добавляет сотрудников и роли, назначает права;  
   - создает услуги и пакеты услуг, формирует график работы и расписание мастеров;  
   - отслеживает все сделки, лиды, клиентов, отчеты и платежи;  
   - подключает интеграции: формы на сайте, Instagram, Google Calendar и платежные системы.  

2. **Менеджер / сотрудник салона**  
   - работает с клиентами и лидами (назначает встречи, звонки, отправляет письма/смс);  
   - ведет сделки, отслеживает этапы воронки;  
   - использует кастомные поля для более точного учета данных по клиенту.  

3. **Клиент**  
   - посещает маркетинговый сайт, читает статьи и новости по теме тату;  
   - оставляет заявку/контакт → лид автоматически попадает в CRM;  
   - регистрируется в личном кабинете, просматривает свои записи, историю взаимодействий, портфолио мастеров.  

4. **AI / SEO система**  
   - собирает семантическое ядро, генерирует статьи, оптимизирует страницы;  
   - мониторит позиции в поиске (SerpAPI), создает рекомендации для редактора или автоматически обновляет контент;  
   - поддерживает автоматические переводы на DE/EN/ES/FR/NL.  

---

## Целевая аудитория

- Владельцы и менеджеры тату‑студий в Европе;  
- Частные клиенты, ищущие тату‑мастеров, портфолио, советы по уходу;  
- SEO‑оптимизаторы и маркетологи, которые смогут использовать автоматизацию для роста трафика.  

---

## Ключевые требования (высокоуровневые)

### 1. Технологии и архитектура
- Monorepo: frontend Next.js + backend NestJS;  
- Возможность масштабирования на несколько порталов;  
- Поддержка SSR/ISR/SSG для SEO страниц.  

### 2. Безопасность и GDPR/DSGVO
- Надёжная аутентификация и мульти-пользовательские порталы;  
- Управление сессиями, 2FA, email verification;  
- Удаление данных по запросу, хранение согласий;  
- Audit logs для действий сотрудников.  

### 3. CRM функционал
- Управление сущностями: Client, Lead, Deal, Appointment;  
- Кастомные поля для порталов;  
- Отчеты, интеграции с внешними системами, формы для лидов;  
- Поддержка календаря, напоминаний, уведомлений.  

### 4. Маркетинг и SEO
- Автоматическая генерация семантического ядра;  
- AI-генерация контента (статьи, metadata, FAQ schema, OG);  
- Мониторинг SERP, рекомендации по улучшению контента;  
- Переводы и локализация под целевые страны.  

### 5. Платежи и биллинг
- Интеграция EU-ready платежных систем (Stripe, Mollie, PayPal);  
- Подписки, инвойсы, возвраты, VAT handling;  
- Безопасная обработка платежей (PCI DSS).  

### 6. UI/UX и клиентский опыт
- Next.js + Tailwind/shadcn UI;  
- Мобильная адаптивность, быстрая загрузка, accessibility;  
- Личный кабинет для клиентов и CRM портал для сотрудников.  

---

## Дополнительно (масштабируемость и будущие апдейты)

- Возможность подключения маркетинговых и AI‑фич по шагам, без остановки основной CRM;  
- Возможность интеграции с другими сервисами (Instagram, Facebook, Email, Google Ads);  
- Возможность расширения CRM функционала: marketplace для мастеров, multi-studio support.
